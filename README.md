# RSpec Exercise

Includes Rspec tests for previous projects:
* Caesar cipher
* Enumberable methods (without using inbuild Ruby methods)
* Tic Tac Toe


## Tests

Navigate to each of the respective folders, e.g. tictactoe/spec
```
cd tictactoe/spec
rake spec
```
Tests should be successful

* Ruby
